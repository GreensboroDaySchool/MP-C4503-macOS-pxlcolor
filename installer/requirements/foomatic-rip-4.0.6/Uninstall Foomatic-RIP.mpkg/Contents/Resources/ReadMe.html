Uninstaller for Foomatic-RIP

Description

The Foomatic-RIP uninstaller safely removes all of the Foomatic-RIP software. The Foomatic-RIP uninstaller does not remove Ghostscript, which may be in use by other drivers on your system. If you wish to remove Ghostscript you must use the separate uninstaller for that software.

Using the uninstaller

To begin the uninstall process simply click Continue below; on the following screen select the appropriate volume and click Continue; on the final screen click the Upgrade button.

Please note that although the button says Upgrade you are actually removing the software. If the final button says Install, rather than Upgrade, then running the uninstaller will do nothing to your system.

Mac OS X 10.5.x (Leopard) and Mac OS X 10.6.x (Snow Leopard) users will always see an Install button. The uninstaller will still perform properly.
