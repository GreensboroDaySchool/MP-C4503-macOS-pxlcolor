Foomatic-RIP
released March 28, 2011

Software Requirements

System software: Mac OS X Panther (10.3.x) or later.

Additional software: This software requires Ghostscript and at least one Foomatic PPD.

Description

On Mac OS X, foomatic-rip is a native CUPS filter that coordinates the printing process for drivers built for the Foomatic printing system.

Release Notes

  • This release also includes a backend handler (beh). A file describing the use of the backend handler is included on this disk image.

Known Issues

Certain PPD files for PostScript printers obtained from the OpenPrinting.org site will cause Foomatic-RIP to crash. Before using a PostScript PPD with Foomatic-RIP, examine the PPD file for the line:

*cupsFilter:	"application/vnd.cups-pdf 0 foomatic-rip"

If that line is found, it should be changed to: 

*%cupsFilter:	"application/vnd.cups-pdf 0 foomatic-rip"

Installation and Set Up

Note for upgrading users: You may safely install this version of the software over any other previous version.

Installing: Simply run the installer (administrator privileges will be required).

Support

User-to-user support for printing under Mac OS X is available in our Mac OS X forum. Other sources for help are Apple's Discussion forums (Snow Leopard printing, Leopard printing, Tiger printing, and Panther printing)
